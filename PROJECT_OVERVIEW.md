## How It Works

### One-Liner
**Automated AI system that monitors news + generates marketing angles/headlines per GEO, 2x per week**

### Three Steps

```
1. AGGREGATE NEWS
   RSS feeds → Deduplicate → Filter by date/GEO
   Result: 15-20 news items per GEO

2. PROCESS WITH AI
   Haiku: Classify (fast/cheap)
   Sonnet: Generate angles → headlines (quality)
   Sonnet: Assess risks (compliance)
   Result: 20-30 angles, 30-50 headlines

3. DELIVER REPORT
   JSON for Airtable
   Text for humans
   Slack/Telegram notifications
   Result: Ready-to-test creative pack
```

### What It Outputs

For each GEO (Russia, Ukraine, Belarus):

```
📊 Report Structure:

Block 1: Raw News (10-20)
├─ Headlines with sources
├─ Categories (economy, politics, celebrity, etc)
├─ Emotional triggers (money, fear, opportunity)
└─ Urgency classification

Block 2: Marketing Angles (20-30)
├─ News → angle → offer connection
├─ Target audience pain
├─ Creative type recommendation
└─ Testing priority (A/B/C)

Block 3: Ad Headlines (30-50)
├─ Grouped by angle
├─ Multiple formats (question, shock, number, quote, intrigue)
└─ Character counts

Block 4: Testing Recommendations
├─ Top-5 angles with justification
├─ 3 best headlines per angle
└─ Ready-to-test package

Block 5: Risk Assessment
├─ Legal risks
├─ Platform ban risk
├─ Audience negativity risk
├─ Reputation risk
└─ Content expiry date

Block 6: Urgency Summary
├─ 🔥 Urgent (test within 48h)
└─ ⏳ Eternal themes
```

## Key Decisions Made

### 1. **Haiku for Classification, Sonnet for Generation**
- ✅ 70% cheaper than Sonnet-only
- ✅ Faster iteration
- ✅ Same quality output
- ❌ Slightly different responses (not critical for marketing)

### 2. **Modular Architecture**
- ✅ Easy to modify prompts
- ✅ Reusable components
- ✅ Simple to add new news sources
- ✅ Clear separation of concerns

### 3. **JSON Output Format**
- ✅ Machine-readable for Airtable
- ✅ Preserves all metadata
- ✅ Easy to integrate with other tools
- ✅ Debuggable

### 4. **Multi-Channel Notifications**
- ✅ Slack: For team visibility
- ✅ Telegram: For instant mobile alerts
- ✅ JSON: For structured data
- ✅ All optional - works without any

### 5. **No External Database Required**
- ✅ Works with just JSON files
- ✅ Optional Airtable for advanced tracking
- ✅ No DevOps overhead
- ✅ Start local, scale later

## Troubleshooting

### "demo.py shows nothing"
```bash
$env:PYTHONIOENCODING='utf-8'  # Windows
export PYTHONIOENCODING=utf-8   # Mac/Linux
python demo.py
```

### "main.py fails with API key error"
1. Get key: https://console.anthropic.com/
2. Edit `.env`: `ANTHROPIC_API_KEY=sk-ant-...`
3. Run: `python main.py`

### "No news found"
- Check internet connection
- Verify RSS feeds are accessible
- Try with different GEO in `config.py`

### "Want to modify prompts?"
Edit the prompt strings in `llm_processor.py` - they're clear and easy to understand.
